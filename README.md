## Shadow-Metrics
#### 1. Descrição do Projeto
O Shadow Metrics é um projeto acadêmico desenvolvido para implementar uma solução completa de monitoramento utilizando o ecossistema Zabbix. O sistema realiza a coleta de dados de ativos de rede e servidores, permitindo a gestão proativa de recursos através de triggers personalizadas e dashboards dinâmicos.

#### 4. Implementação
##### Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:
```env
  PORT=3000
```

##### Instalação e Execução
```bash
  # Instale as dependecias
  npm install

  # Inicie a aplicação
  npm run dev 
```
